# Smart Home Load Monitor for Ghanaian Homes

JavaFX desktop application simulating current monitoring for 13A socket groups.

## Features
- Real-time current display per appliance and totals
- Socket group status (OK / WARNING / DANGER)
- Overload alerts and load reduction recommendations
- Surge detection
- Invalid sensor handling
- Whole-house limit (main/inverter)
- Energy tracking and cost estimation

## How to Run
Requires JavaFX SDK.

java --module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml Main
